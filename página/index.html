Exemplos de Código

Aritméticos

let soma = 10 + 5;      // 15
let expo = 2 ** 3;     // 8 (2 ao cubo)
let resto = 10 % 3;    // 1 (sobra da divisão)

Comparação

console.log(5 == "5");  // true (valor igual)
console.log(5 === "5"); // false (tipo diferente)
console.log(10 > 5);    // true

Lógicos

let user = true;
let admin = false;

console.log(user && admin); // false (AND)
console.log(user || admin); // true (OR)

Atribuição

let x = 10;
x += 5; // Resultado: 15
x *= 2; // Resultado: 30
